There are no browsers that fully support the ES6 features.However, we can convert the ES6 code to the ES5 code by using the transpilation. I used Babel which is used to convert the ES6 code to ES5 code as part of the build process.
There are many difference between ES6 and ES5.ES6 classes can be easily implemented over the prototype based object oriented pattern. In ES5,reserved words can use as property names.There is also no "class" keyword in ES5.
We'll use to implement class-like functionalities using "functions".We can use prototype to add/extend new properties and i did the same with the functions. In ES5, there is no extends that can extend all of the properties, no constructor for initialization or super to call the constructor of the base class.
We're binding the properties with "this".
